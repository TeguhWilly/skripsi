Petunjuk Umum:
  • Bila anda sudah pernah mendaftar, maka silakan login dengan menggunakan username dan password ketika mendaftar dulu
  • Bila belum silakan lakukan pendaftaran pada menu sebelah kiri
  • Setelah melakukan pendaftaran anda langsung dapat login untuk mengikuti tes
  • Semua pendaftar baru adalah sebagai user siswa
  • Untuk mengubah user level menjadi user guru, silakan hubungi administrator
Petunjuk Input Text Uji :
  • Menu Tes_Uraian - Buat_tes untuk membuat tes plagiat baru
  • Untuk menambahkan soal pilih Tes_Uraian - Input_text_uji, pada tabel daftar tes klik link input pada kolom lihat tes setelah itu klik tombol tambah soal
  • Menu Tes_Uraian - Tambah_kata digunakan untuk menambahkan kata baru
  • Menu Tes_uraian - Daftar_kata untuk melihat kata yang tersedia dan menambahkan sinonim kata
Petunjuk Input Text Latih :
  • Klik menu Tes_uraian - Input_text_latih - klik link input
  • Setelah itu akan muncul daftar soal dalam cek kemiripan teks tersebut, klik simpan bila telah selesai
